# 📒 店铺记账本

一个轻量的店铺记账 PWA 应用，运行在 iPhone 上，数据完全存储在手机本地。

---

## 🚀 如何部署到你的 iPhone

### 第一步：创建 GitHub 仓库

1. 打开 [github.com](https://github.com) 并登录（没有账号的话先注册一个，免费）
2. 点击右上角的 **+** 号 → **New repository**
3. 仓库名填：`jizhangben`（或任意英文名）
4. 选择 **Public**（公开）
5. 不要勾选任何初始化选项
6. 点击 **Create repository**

### 第二步：上传文件

将以下 **4 个文件** 拖拽到仓库页面的文件区域：

- `index.html`
- `manifest.json`
- `sw.js`
- `README.md`

在底部填写提交说明（如"初始化记账本"），然后点击 **Commit changes**。

### 第三步：开启 GitHub Pages

1. 在仓库页面点击 **Settings**（设置）
2. 左侧菜单点击 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. **Branch** 选择 **main**，文件夹选 **/ (root)**
5. 点击 **Save**
6. 等待 1-2 分钟，页面上会显示：
   > Your site is live at `https://你的用户名.github.io/jizhangben/`

### 第四步：安装到 iPhone

1. 用 **Safari** 打开上面的网址（必须用 Safari，其他浏览器不行）
2. 点击底部的 **分享按钮**（方框带向上箭头图标）
3. 向下滑动，找到并点击 **"添加到主屏幕"**
4. 名称改为 **"记账本"**，点击右上角 **添加**
5. 桌面上会出现记账本图标，点击即可使用 🎉

## 📱 功能介绍

| 功能 | 说明 |
|------|------|
| 🏠 **首页** | 每日收支概览、低库存提醒、快捷记账入口 |
| 📦 **库存** | 管理商品（添加/编辑/删除）、查看库存量 |
| ✏️ **记账** | 记录进货和卖出，自动计算利润、自动更新库存 |
| 📅 **日历** | 浏览任意日期的交易记录 |
| 📊 **统计** | 按日/周/月/年查看盈亏报表，支持导出 Excel |

---

## ⚠️ 注意事项

- **数据在手机本地**，清除 Safari 浏览数据会导致数据丢失，请在清除前先导出 Excel 备份
- **换手机前**务必导出 Excel 备份数据
- **不支持 iCloud 同步**，数据只在当前手机上
- 如果页面显示异常，可以尝试**下拉刷新**

---

## 🔄 如何更新

当你需要更新功能时，重新上传新的 `index.html` 等文件到 GitHub 仓库即可。iPhone 上的 App 会自动更新（首次打开时可能需要联网）。

---

## 📤 数据备份

在 **统计** 页面底部，选择起止日期后点击 **导出 Excel**，文件会下载到手机。你可以用 WPS 或 Numbers 打开查看。
